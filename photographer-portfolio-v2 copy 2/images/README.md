# Photographer Portfolio Template - Images

This folder contains all the images used in the photographer portfolio template.

## Required Images

### Essential Images (Required for template to work properly):

1. **logo.png** (200x50px, transparent PNG)
   - Professional photography logo
   - Should be transparent background
   - High quality, crisp design

2. **hero-bg.jpg** (1920x1080px, high quality)
   - Hero section background image
   - Should be professional photography related
   - High resolution, good quality

3. **about-photographer.jpg** (800x600px)
   - Professional headshot of the photographer
   - High quality portrait
   - Professional appearance

### Portfolio Gallery Images (6 images):

4. **wedding-1.jpg** (800x600px)
   - Wedding photography sample
   - High quality, professional shot

5. **portrait-1.jpg** (800x600px)
   - Portrait photography sample
   - Professional portrait example

6. **family-1.jpg** (800x600px)
   - Family photography sample
   - Natural, candid family moment

7. **event-1.jpg** (800x600px)
   - Event photography sample
   - Corporate or special event

8. **commercial-1.jpg** (800x600px)
   - Commercial photography sample
   - Business or product photography

9. **landscape-1.jpg** (800x600px)
   - Landscape photography sample
   - Beautiful natural or urban landscape

## Image Guidelines

- **Format**: JPG for photos, PNG for logo
- **Quality**: High resolution, professional quality
- **Optimization**: Compress for web but maintain quality
- **Consistency**: All portfolio images should have similar style/quality
- **Rights**: Ensure you have rights to use all images

## Customization

Users can replace any of these images with their own:
- Click on any image in the GrapesJS editor
- Upload their own image
- The template will automatically adjust

## File Naming

Keep the exact filenames as listed above for the template to work properly. If you change filenames, you'll need to update the HTML references as well.
